# Gaming Journal Club Client

React/Vite frontend for Gaming Journal Club.

For the full MVP setup, demo login, AI SYNC smoke test, and known issues, read the root `README.md`.

## Local Commands

```bash
npm install
npm run dev -- --host 127.0.0.1
npm run lint
npm run build
```

The app expects the NestJS API at:

```text
http://localhost:3000
```

Override it with `VITE_API_URL` when needed.
